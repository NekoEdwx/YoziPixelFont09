# YoziPixelFont09 (v0.0.1)
This is a project for pixel font files called "YoziPixel" created by NekoYozi. This font is characterized by its high readability, and the Chinese characters conform to Chinese writing conventions (mostly).

## Usage
1. Download the latest `.ttf` file from the [Releases] page.
2. Double‑click the font file and click "Install".
3. Use it in any application that supports custom fonts.

## Contribution Guidelines
We very welcome suggestions for font modifications or your own drawn glyphs submitted via Issue.  
If you would like to contribute new characters or improve existing glyphs, please submit the corresponding character’s text along with a PNG image.  
If you are an artist, please provide an unscaled image with a transparent background, with each glyph sized 9 pixels (including the edge will be 11 pixels) to clearly show the glyph details. For a large number of characters, you can combine them into one image), or alternatively a TTF file containing only that character (e.g., a single‑character font file exported from a font editing tool.  
**Please do not submit the complete font file containing other unmodified characters, so that we can efficiently review and integrate your contributions！**  
Thank you very much! :>
